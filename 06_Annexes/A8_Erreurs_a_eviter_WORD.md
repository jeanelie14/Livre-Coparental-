ANNEXE A8 — ERREURS À ÉVITER

─────────────────────────────────────────────────────────────────────────────

INSTRUCTIONS

Cette annexe identifie les erreurs les plus courantes dans chaque partie du livre et explique comment les éviter. Apprendre de ces erreurs peut vous aider à éviter des problèmes et à protéger efficacement votre enfant.

─────────────────────────────────────────────────────────────────────────────

LES 3 ERREURS LES PLUS COURANTES PAR PARTIE

PARTIE I : COMPRENDRE LA SITUATION

ERREUR 1 : MINIMISER LA DIFFICULTÉ DE L'AUTRE PARENT

Pourquoi elle est problématique :
• Vous pouvez sous-estimer les risques pour votre enfant
• Vous pouvez ne pas prendre les mesures de protection nécessaires
• Vous pouvez être pris au dépourvu par des comportements problématiques

Comment l'éviter :
• Évaluez honnêtement la situation
• Identifiez les comportements problématiques
• Prenez les mesures de protection appropriées
• Consultez un professionnel si nécessaire

ERREUR 2 : ESSAYER DE "RÉPARER" L'AUTRE PARENT

Pourquoi elle est problématique :
• Vous ne pouvez pas changer l'autre parent
• Vous perdez du temps et de l'énergie
• Vous pouvez mettre votre enfant en danger en attendant que l'autre parent change

Comment l'éviter :
• Acceptez que vous ne pouvez pas changer l'autre parent
• Concentrez-vous sur la protection de votre enfant
• Établissez des limites claires
• Protégez-vous et votre enfant

ERREUR 3 : NIER L'IMPACT SUR L'ENFANT

Pourquoi elle est problématique :
• Vous pouvez ne pas voir les signes que l'enfant a besoin d'aide
• Vous pouvez ne pas prendre les mesures nécessaires pour protéger l'enfant
• L'enfant peut souffrir sans soutien

Comment l'éviter :
• Surveillez les signes d'impact sur l'enfant
• Écoutez l'enfant
• Consultez un professionnel pour l'enfant si nécessaire
• Prenez les mesures nécessaires pour protéger l'enfant

─────────────────────────────────────────────────────────────────────────────

PARTIE II : SÉCURITÉ

ERREUR 1 : NE PAS DOCUMENTER LES INCIDENTS

Pourquoi elle est problématique :
• Vous n'avez pas de preuves si vous devez aller en cour
• Vous pouvez oublier des détails importants
• Vous pouvez avoir du mal à démontrer les problèmes

Comment l'éviter :
• Documentez tous les incidents (voir Annexe 2.B)
• Gardez une trace de toutes les communications
• Conservez tous les documents importants
• Utilisez un journal d'incidents

ERREUR 2 : ATTENDRE TROP LONGTEMPS AVANT DE SIGNALER À LA DPJ

Pourquoi elle est problématique :
• L'enfant peut être en danger pendant que vous attendez
• La situation peut s'aggraver
• Vous pouvez perdre des opportunités de protection

Comment l'éviter :
• Signalez immédiatement si l'enfant est en danger
• Ne minimisez pas les risques
• Consultez un professionnel si vous avez des doutes
• Agissez rapidement pour protéger l'enfant

ERREUR 3 : NE PAS AVOIR DE PLAN DE SÉCURITÉ

Pourquoi elle est problématique :
• Vous pouvez ne pas savoir quoi faire en cas d'urgence
• Vous pouvez être pris au dépourvu
• Vous pouvez mettre votre enfant en danger

Comment l'éviter :
• Créez un plan de sécurité (voir Annexe 2.A)
• Préparez un sac d'urgence
• Identifiez les lieux sûrs
• Pratiquez votre plan

─────────────────────────────────────────────────────────────────────────────

PARTIE III : COMMUNICATION

ERREUR 1 : RÉAGIR AUX PROVOCATIONS

Pourquoi elle est problématique :
• Vous pouvez escalader les conflits
• Vous pouvez donner du pouvoir à l'autre parent
• Vous pouvez nuire à votre cas si vous allez en cour

Comment l'éviter :
• Ne répondez pas aux provocations
• Restez calme et objectif
• Utilisez des techniques de communication stratégique
• Consultez un professionnel si nécessaire

ERREUR 2 : COMMUNIQUER PAR TÉLÉPHONE OU EN PERSONNE

Pourquoi elle est problématique :
• Les conversations ne sont pas documentées
• Vous pouvez être manipulé ou intimidé
• Vous pouvez dire des choses que vous regretterez

Comment l'éviter :
• Communiquez par écrit (courriel, application)
• Évitez les conversations téléphoniques conflictuelles
• Documentez toutes les communications
• Utilisez des applications de coparentalité

ERREUR 3 : CRITIQUER L'AUTRE PARENT DEVANT L'ENFANT

Pourquoi elle est problématique :
• Vous pouvez nuire à la relation enfant-parent
• Vous pouvez créer de la confusion et de la détresse chez l'enfant
• Vous pouvez être accusé d'aliénation parentale

Comment l'éviter :
• Ne critiquez jamais l'autre parent devant l'enfant
• Protégez l'enfant des conflits
• Concentrez-vous sur le bien-être de l'enfant
• Parlez avec un professionnel si vous avez besoin de soutien

─────────────────────────────────────────────────────────────────────────────

PARTIE IV : LÉGAL

ERREUR 1 : NE PAS CONSULTER UN AVOCAT

Pourquoi elle est problématique :
• Vous pouvez manquer des aspects importants
• Vous pouvez perdre des droits sans le savoir
• Vous pouvez créer des problèmes à long terme

Comment l'éviter :
• Consultez un avocat même pour un divorce à l'amiable
• Utilisez l'aide juridique si vous êtes admissible
• Consultez Éducaloi pour de l'information juridique gratuite
• Ne signez rien sans consulter un avocat

ERREUR 2 : ACCEPTER UNE ENTENTE INÉQUITABLE PAR PEUR

Pourquoi elle est problématique :
• Vous pouvez avoir des difficultés financières
• Vous pouvez perdre des droits importants
• Vous pouvez avoir besoin de modifier l'entente plus tard

Comment l'éviter :
• Ne signez pas sous la pression
• Consultez un avocat avant de signer
• Prenez le temps de réfléchir
• Rappelez-vous que vous avez des droits

ERREUR 3 : NE PAS RÉVÉLER TOUS VOS REVENUS

Pourquoi elle est problématique :
• L'entente peut être annulée si la dissimulation est découverte
• Vous pouvez être poursuivi pour fraude
• Vous pouvez perdre la garde si vous mentez au tribunal

Comment l'éviter :
• Soyez transparent sur tous vos revenus et biens
• Fournissez tous les documents demandés
• Consultez un avocat si vous avez des doutes
• Ne cachez rien

─────────────────────────────────────────────────────────────────────────────

PARTIE V : AVENIR

ERREUR 1 : FORCER L'ENFANT À ACCEPTER UN NOUVEAU PARTENAIRE

Pourquoi elle est problématique :
• L'enfant peut se sentir trahi ou confus
• L'enfant peut rejeter le nouveau partenaire
• Vous pouvez nuire à la relation enfant-parent

Comment l'éviter :
• Présentez le nouveau partenaire progressivement
• Respectez le rythme de l'enfant
• Ne forcez pas l'enfant à accepter
• Consultez un professionnel si nécessaire

ERREUR 2 : NE PAS MAINTENIR LES ROUTINES ET LES RITUELS

Pourquoi elle est problématique :
• L'enfant peut perdre son sentiment de sécurité
• L'enfant peut avoir des difficultés d'adaptation
• Vous pouvez nuire au bien-être de l'enfant

Comment l'éviter :
• Maintenez les routines quotidiennes
• Créez de nouveaux rituels si nécessaire
• Assurez la stabilité pour l'enfant
• Adaptez les routines selon les besoins

ERREUR 3 : NE PAS RÉPARER LA RELATION PARENT-ENFANT

Pourquoi elle est problématique :
• La relation peut se détériorer davantage
• L'enfant peut souffrir émotionnellement
• Vous pouvez perdre votre relation avec l'enfant

Comment l'éviter :
• Identifiez les dommages à la relation
• Prenez des mesures pour réparer la relation
• Consultez un professionnel si nécessaire
• Soyez patient et persévérant

─────────────────────────────────────────────────────────────────────────────

ERREURS GÉNÉRALES À ÉVITER

ERREUR : AGIR SEUL(E) SANS SOUTIEN

Pourquoi elle est problématique :
• Vous pouvez être submergé(e) par les défis
• Vous pouvez prendre de mauvaises décisions
• Vous pouvez nuire à votre santé mentale

Comment l'éviter :
• Cherchez du soutien (famille, amis, professionnels)
• Accédez aux ressources de soutien
• Ne restez pas isolé(e)
• Prenez soin de vous

ERREUR : IGNORER VOTRE SANTÉ MENTALE

Pourquoi elle est problématique :
• Vous ne pouvez pas protéger efficacement votre enfant si vous êtes épuisé(e)
• Vous pouvez prendre de mauvaises décisions
• Vous pouvez nuire à votre relation avec l'enfant

Comment l'éviter :
• Prenez soin de votre santé mentale
• Cherchez du soutien professionnel si nécessaire
• Pratiquez des activités de bien-être
• Maintenez un réseau de soutien

ERREUR : NE PAS PROTÉGER VOTRE ENFANT

Pourquoi elle est problématique :
• L'enfant peut être en danger
• L'enfant peut souffrir émotionnellement
• Vous pouvez perdre la garde si vous ne protégez pas l'enfant

Comment l'éviter :
• Priorisez toujours la sécurité de l'enfant
• Documentez les incidents
• Signalez à la DPJ si nécessaire
• Consultez un professionnel si vous avez des préoccupations

COMMENT ÉVITER CES ERREURS

ÉDUCATION

Informez-vous :
• Lisez ce livre complètement
• Consultez les ressources disponibles
• Informez-vous sur vos droits et obligations

SOUTIEN PROFESSIONNEL

Consultez des professionnels :
• Avocat pour les aspects légaux
• Psychologue pour le soutien émotionnel
• Travailleur social pour le soutien pratique
• Médiateur pour la résolution de conflits

DOCUMENTATION

Documentez tout :
• Gardez une trace de toutes les communications
• Documentez tous les incidents
• Conservez tous les documents importants

PROTECTION

Protégez-vous et votre enfant :
• Créez un plan de sécurité
• Établissez des limites claires
• Signalez les situations dangereuses
• Consultez un professionnel si nécessaire
