ANNEXE A7 — FAQ (FOIRE AUX QUESTIONS)

─────────────────────────────────────────────────────────────────────────────

INSTRUCTIONS

Cette FAQ répond aux questions les plus fréquentes sur la séparation et la coparentalité au Québec. [IMPORTANT] Ces réponses sont générales. Consultez toujours un avocat pour des conseils spécifiques à votre situation.

─────────────────────────────────────────────────────────────────────────────

QUESTIONS FRÉQUENTES

"PUIS-JE DÉMÉNAGER AVEC MON ENFANT ?"

Réponse : Cela dépend de votre situation et de vos arrangements de garde.

Si vous avez la garde principale :
• Vous pouvez généralement déménager dans la même région sans autorisation
• Pour un déménagement important (changement de région, province, pays), vous devez généralement obtenir l'autorisation de l'autre parent ou du tribunal

Si vous avez la garde partagée :
• Vous devez généralement obtenir l'autorisation de l'autre parent ou du tribunal pour déménager
• Le tribunal considérera l'impact sur l'enfant et la relation avec l'autre parent

Que faire :
• Consultez un avocat avant de déménager
• Communiquez avec l'autre parent si possible
• Documentez vos raisons de déménager
• Préparez-vous à justifier le déménagement devant le tribunal si nécessaire

[IMPORTANT] Déménager sans autorisation peut avoir des conséquences légales graves.

─────────────────────────────────────────────────────────────────────────────

"QUE FAIRE SI L'AUTRE PARENT NE PAIE PAS LA PENSION ?"

Réponse : Vous avez plusieurs recours disponibles.

Actions immédiates :
1. Contactez l'autre parent pour clarifier la situation
2. Documentez les non-paiements
3. Consultez un avocat

Recours disponibles :
1. Revenu Québec : Revenu Québec peut recouvrer la pension alimentaire (1-800-267-7829)
2. Tribunal : Vous pouvez demander au tribunal de faire exécuter la pension
3. Saisie : Vous pouvez demander une saisie sur le salaire ou les comptes bancaires

Documentation nécessaire :
• Copie de l'entente ou du jugement
• Preuve des non-paiements
• Documentation des tentatives de communication

Ressources :
• Revenu Québec : 1-800-267-7829
• Voir Chapitre 13 pour plus de détails

─────────────────────────────────────────────────────────────────────────────

"MON ENFANT REFUSE DE VOIR L'AUTRE PARENT, QUE FAIRE ?"

Réponse : Cela dépend de la raison du refus et de l'âge de l'enfant.

Si l'enfant est en danger :
• Agissez immédiatement pour protéger l'enfant
• Contactez la DPJ (1-800-461-9331, gratuit et confidentiel, 24/7) si nécessaire
• Consultez un avocat pour obtenir des mesures de protection

Si l'enfant refuse sans raison valable :
• Parlez avec l'enfant pour comprendre la situation
• Consultez un psychologue pour l'enfant
• Documentez la situation
• Consultez un avocat si nécessaire

Si l'enfant est manipulé (aliénation parentale) :
• Documentez les comportements d'aliénation
• Consultez un psychologue spécialisé en aliénation parentale
• Consultez un avocat pour obtenir des mesures de protection
• Voir Chapitre 21 pour plus de détails

[IMPORTANT] Ne forcez pas l'enfant à voir l'autre parent si l'enfant est en danger.

─────────────────────────────────────────────────────────────────────────────

"PUIS-JE CHANGER L'ÉCOLE SANS L'ACCORD DE L'AUTRE PARENT ?"

Réponse : Généralement non, si vous avez la garde partagée ou si l'autre parent a des droits de décision.

Si vous avez la garde exclusive :
• Vous pouvez généralement changer l'école sans l'accord de l'autre parent
• Cependant, il est recommandé d'informer l'autre parent

Si vous avez la garde partagée :
• Vous devez généralement obtenir l'accord de l'autre parent
• Si vous ne pouvez pas vous entendre, vous pouvez demander au tribunal de décider

Que faire :
• Consultez votre entente de séparation ou votre jugement
• Communiquez avec l'autre parent
• Consultez un avocat si nécessaire

[IMPORTANT] Changer l'école sans autorisation peut avoir des conséquences légales.

─────────────────────────────────────────────────────────────────────────────

"COMMENT GÉRER LES VACANCES ?"

Réponse : Les vacances doivent être planifiées à l'avance et partagées selon vos arrangements.

Planification :
• Planifiez les vacances plusieurs mois à l'avance
• Communiquez avec l'autre parent
• Établissez un calendrier annuel des vacances

Partage :
• Partagez les vacances selon vos arrangements (alternance, partage, etc.)
• Respectez les arrangements convenus
• Soyez flexible si possible

Communication :
• Communiquez vos plans avec l'autre parent
• Donnez un avis préalable suffisant
• Respectez les plans de l'autre parent

Voir Chapitre 20 pour plus de détails.

─────────────────────────────────────────────────────────────────────────────

"QUE FAIRE SI L'AUTRE PARENT NE RESPECTE PAS L'ENTENTE ?"

Réponse : Vous avez plusieurs options selon la gravité de la violation.

Actions immédiates :
1. Documentez les violations
2. Communiquez avec l'autre parent (si sécuritaire)
3. Consultez un avocat

Recours disponibles :
1. Médiation : Si les violations sont mineures, la médiation peut aider
2. Modification de l'entente : Si les violations sont importantes, vous pouvez demander une modification
3. Exécution par le tribunal : Vous pouvez demander au tribunal de faire exécuter l'entente

Documentation nécessaire :
• Copie de l'entente ou du jugement
• Preuve des violations
• Documentation des tentatives de communication

[IMPORTANT] Ne violez pas l'entente vous-même en représailles.

─────────────────────────────────────────────────────────────────────────────

"PUIS-JE ENREGISTRER LES CONVERSATIONS ?"

Réponse : La législation québécoise sur l'enregistrement des conversations est complexe.

Règles générales :
• Vous pouvez généralement enregistrer une conversation si vous y participez
• Cependant, l'utilisation de l'enregistrement peut être limitée selon le contexte
• L'enregistrement à l'insu de l'autre personne peut être illégal dans certains cas

Que faire :
• Consultez un avocat avant d'enregistrer des conversations
• Documentez les conversations par écrit (courriel, SMS) quand possible
• Utilisez un journal d'incidents (voir Annexe 2.B)

[IMPORTANT] Les lois sur l'enregistrement peuvent varier. Consultez un avocat pour des conseils spécifiques.

─────────────────────────────────────────────────────────────────────────────

"COMMENT PROTÉGER MON ENFANT DES RÉSEAUX SOCIAUX DE L'AUTRE PARENT ?"

Réponse : Vous pouvez prendre plusieurs mesures pour protéger votre enfant.

Mesures immédiates :
1. Parlez avec l'autre parent au sujet des réseaux sociaux
2. Établissez des règles communes concernant les réseaux sociaux
3. Surveillez la présence en ligne de votre enfant

Mesures de protection :
1. Limitez l'accès aux informations personnelles
2. Utilisez des paramètres de confidentialité stricts
3. Éduquez votre enfant sur la sécurité en ligne
4. Surveillez l'activité en ligne de votre enfant

Si l'autre parent publie des informations problématiques :
1. Documentez les publications problématiques
2. Communiquez avec l'autre parent
3. Consultez un avocat si nécessaire

Voir Chapitre 9 pour plus de détails.

AUTRES QUESTIONS FRÉQUENTES

"PUIS-JE MODIFIER L'ENTENTE DE SÉPARATION ?"

Réponse : Oui, si les circonstances ont changé.

Quand modifier :
• Changement de revenus significatif
• Changement de garde
• Changement des besoins de l'enfant
• Déménagement
• Autres changements de circonstances

Comment modifier :
• Par entente avec l'autre parent
• Par demande au tribunal si vous ne pouvez pas vous entendre

Voir Chapitre 13 pour plus de détails.

─────────────────────────────────────────────────────────────────────────────

"QUE FAIRE SI L'AUTRE PARENT ME HARCELLE ?"

Réponse : Vous pouvez obtenir une ordonnance de protection.

Actions immédiates :
1. Documentez le harcèlement
2. Contactez la police si nécessaire
3. Consultez un avocat pour obtenir une ordonnance de protection

Ordonnance de protection :
• Peut interdire le contact
• Peut interdire l'approche
• Peut inclure d'autres mesures de protection

Voir Chapitre 11 pour plus de détails.

─────────────────────────────────────────────────────────────────────────────

"PUIS-JE VOYAGER AVEC MON ENFANT SANS L'ACCORD DE L'AUTRE PARENT ?"

Réponse : Cela dépend de votre situation et de la destination.

Voyage au Canada :
• Généralement permis sans autorisation
• Cependant, il est recommandé d'informer l'autre parent

Voyage à l'étranger :
• Généralement nécessite l'autorisation de l'autre parent
• Peut nécessiter une autorisation du tribunal
• Peut nécessiter des documents supplémentaires (passeport, autorisation de voyage)

Que faire :
• Consultez votre entente de séparation ou votre jugement
• Communiquez avec l'autre parent
• Consultez un avocat si nécessaire

[IMPORTANT] Voyager sans autorisation peut avoir des conséquences légales graves, surtout à l'étranger.

"COMMENT CALCULER LA PENSION ALIMENTAIRE ?"

Réponse : La pension alimentaire est calculée selon des tables officielles.

Tables de calcul :
• Tables québécoises (si les deux parents résident au Québec)
• Tables fédérales (si un parent réside dans une autre province)

Facteurs considérés :
• Revenus des deux parents
• Nombre d'enfants
• Arrangements de garde
• Autres facteurs

Calculateurs :
• Revenu Québec : revenuquebec.ca
• Éducaloi : educaloi.qc.ca

Voir Chapitre 13 pour plus de détails.

"QUE FAIRE SI L'AUTRE PARENT UTILISE L'ENFANT COMME MESSAGER ?"

Réponse : Protégez l'enfant de cette situation.

Actions :
1. Ne participez pas à la communication par l'enfant
2. Communiquez directement avec l'autre parent
3. Protégez l'enfant de cette situation
4. Documentez les incidents
5. Consultez un avocat si nécessaire

Voir Chapitre 21 pour plus de détails.

"COMMENT GÉRER LES NOUVEAUX PARTENAIRES ?"

Réponse : Intégrez les nouveaux partenaires progressivement et avec soin.

Quand présenter :
• Attendez que la relation soit stable (généralement 6-12 mois)
• Considérez l'âge et la situation de l'enfant

Comment présenter :
• Présentez progressivement
• Établissez des limites claires
• Communiquez avec l'autre parent

Voir Chapitre 16 pour plus de détails.

"QUE FAIRE SI JE SUIS VICTIME DE VIOLENCE ?"

Réponse : Agissez immédiatement pour votre sécurité.

Actions immédiates :
1. Appelez le 9-1-1 si vous êtes en danger immédiat
2. Quittez la situation dangereuse
3. Contactez un refuge ou un service d'aide aux victimes
4. Documentez les incidents

Ressources :
• 9-1-1 : Urgence
• Ligne d'écoute violence conjugale : 1-800-363-9010
• DPJ : 1-800-461-9331 (pour faire un signalement, gratuit et confidentiel, 24/7)
• Refuges pour femmes : Recherchez dans votre région

Voir Chapitres 4 et 5 pour plus de détails.
