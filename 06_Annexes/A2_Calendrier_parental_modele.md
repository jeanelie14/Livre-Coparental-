# Annexe A2 — Calendrier parental modèle

---

## Instructions d'utilisation

Ce calendrier parental vous aide à planifier et à organiser les arrangements de garde et les événements spéciaux. Utilisez-le pour :
- Planifier les horaires de garde
- Organiser les vacances et les fêtes
- Documenter les changements d'horaire
- Communiquer avec l'autre parent

**Comment utiliser :**
1. Choisissez le modèle qui convient à votre situation
2. Imprimez ou copiez le calendrier
3. Remplissez les dates et les arrangements
4. Partagez une copie avec l'autre parent
5. Mettez à jour régulièrement

## Exemples selon différents modèles de garde

### Exemple 1 : Garde partagée 50/50 (semaine alternée)

**Horaire :**
- Semaine 1 : L'enfant passe toute la semaine avec Parent 1
- Semaine 2 : L'enfant passe toute la semaine avec Parent 2
- Les semaines alternent ainsi de façon continue

**Transition :**
- Vendredi 18h00 à l'école

**Calendrier mensuel type :**

| Semaine | Lundi | Mardi | Mercredi | Jeudi | Vendredi | Samedi | Dimanche |
|---------|-------|-------|----------|-------|----------|--------|----------|
| **Semaine 1** | P1 | P1 | P1 | P1 | P1 | P1 | P1 |
| **Semaine 2** | P2 | P2 | P2 | P2 | P2 | P2 | P2 |
| **Semaine 3** | P1 | P1 | P1 | P1 | P1 | P1 | P1 |
| **Semaine 4** | P2 | P2 | P2 | P2 | P2 | P2 | P2 |

### Exemple 2 : Garde partagée 50/50 (2-2-3)

**Horaire :**
- Semaine 1 : L'enfant passe lundi-mardi avec Parent 1, mercredi-jeudi avec Parent 2, vendredi-samedi-dimanche avec Parent 1
- Semaine 2 : L'enfant passe lundi-mardi avec Parent 2, mercredi-jeudi avec Parent 1, vendredi-samedi-dimanche avec Parent 2
- Les semaines alternent ainsi

**Transition :**
- Lundi, mercredi, vendredi 18h00 à l'école

**Calendrier mensuel type :**

| Semaine | Lundi | Mardi | Mercredi | Jeudi | Vendredi | Samedi | Dimanche |
|---------|-------|-------|----------|-------|----------|--------|----------|
| **Semaine 1** | P1 | P1 | P2 | P2 | P1 | P1 | P1 |
| **Semaine 2** | P2 | P2 | P1 | P1 | P2 | P2 | P2 |
| **Semaine 3** | P1 | P1 | P2 | P2 | P1 | P1 | P1 |
| **Semaine 4** | P2 | P2 | P1 | P1 | P2 | P2 | P2 |

### Exemple 3 : Garde principale + visites (fins de semaine)

**Horaire :**
- L'enfant vit principalement avec Parent 1
- L'enfant passe les fins de semaine (vendredi soir au dimanche soir) avec Parent 2
- L'enfant passe une soirée en semaine (mardi ou mercredi) avec Parent 2

**Transition :**
- Vendredi 18h00 et dimanche 18h00 à l'école
- Mardi ou mercredi 18h00 à l'école

**Calendrier mensuel type :**

| Semaine | Lundi | Mardi | Mercredi | Jeudi | Vendredi | Samedi | Dimanche |
|---------|-------|-------|----------|-------|----------|--------|----------|
| **Toutes les semaines** | P1 | P1 (P2 soir) | P1 | P1 | P1 (P2 soir) | P2 | P2 |

### Exemple 4 : Garde principale + visites (mi-semaine + fins de semaine)

**Horaire :**
- L'enfant vit principalement avec Parent 1
- L'enfant passe les mardis et jeudis soirs avec Parent 2
- L'enfant passe les fins de semaine (vendredi soir au dimanche soir) avec Parent 2

**Transition :**
- Mardi, jeudi, vendredi, dimanche 18h00 à l'école

**Calendrier mensuel type :**

| Semaine | Lundi | Mardi | Mercredi | Jeudi | Vendredi | Samedi | Dimanche |
|---------|-------|-------|----------|-------|----------|--------|----------|
| **Toutes les semaines** | P1 | P1 (P2 soir) | P1 | P1 (P2 soir) | P1 (P2 soir) | P2 | P2 |

## Conseils pour utiliser le calendrier

### Planification

**Planifiez à l'avance :**
- Planifiez les vacances plusieurs mois à l'avance
- Planifiez les fêtes au début de l'année
- Mettez à jour le calendrier régulièrement

**Communication :**
- Partagez le calendrier avec l'autre parent
- Communiquez les changements dès que possible
- Documentez les changements d'horaire

### Flexibilité

**Soyez flexible :**
- Permettez des ajustements si nécessaire
- Documentez tous les changements
- Respectez les arrangements convenus

### Documentation

**Gardez une trace :**
- Gardez une copie du calendrier
- Documentez les changements d'horaire
- Notez les événements spéciaux
