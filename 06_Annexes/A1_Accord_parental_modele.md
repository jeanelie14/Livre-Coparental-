# Annexe A1 — Modèle d'accord parental

---

## Instructions d'utilisation

Cet accord parental est un modèle que vous pouvez utiliser pour créer votre propre accord. **⚠️ Important :** Même si vous utilisez ce modèle, il est fortement recommandé de consulter un avocat avant de signer l'accord pour vous assurer qu'il est légal et protège vos droits et ceux de votre enfant.

**Comment utiliser ce modèle :**
1. Lisez les deux versions (simplifiée et détaillée)
2. Choisissez la version qui convient à votre situation
3. Remplissez toutes les sections en remplaçant les [brackets] par vos informations
4. Consultez un avocat pour réviser l'accord
5. Signez l'accord avec l'autre parent
6. Faites homologuer l'accord par le tribunal si nécessaire

---

## Instructions de remplissage

### Étape 1 : Préparation

**Avant de remplir l'accord :**
1. Rassemblez tous les documents nécessaires (revenus, actes de naissance, etc.)
2. Discutez avec l'autre parent des arrangements de base
3. Consultez un avocat pour obtenir des conseils
4. Choisissez la version (simplifiée ou détaillée) qui convient à votre situation

### Étape 2 : Remplissage

**Pour chaque section :**
1. Lisez attentivement la section
2. Remplacez tous les [brackets] par vos informations
3. Supprimez les options qui ne s'appliquent pas
4. Ajoutez des sections supplémentaires si nécessaire

**Conseils :**
- Soyez précis et détaillé
- Évitez les ambiguïtés
- Incluez tous les arrangements importants
- Pensez aux situations futures

### Étape 3 : Révision

**Avant de signer :**
1. Relisez l'accord complètement
2. Vérifiez que toutes les sections sont remplies
3. Vérifiez que les informations sont exactes
4. Consultez un avocat pour réviser l'accord
5. Discutez avec l'autre parent des points à clarifier

### Étape 4 : Signature et homologation

**Après révision :**
1. Signez l'accord avec l'autre parent
2. Faites signer par un témoin si nécessaire
3. Faites homologuer l'accord par le tribunal si nécessaire
4. Gardez des copies de l'accord dans un endroit sûr
5. Partagez une copie avec votre avocat
