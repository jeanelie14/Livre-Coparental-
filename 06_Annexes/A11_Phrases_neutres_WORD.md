ANNEXE A11 — PHRASES NEUTRES ET SCRIPTS POUR SMS ET COURRIEL

─────────────────────────────────────────────────────────────────────────────

INSTRUCTIONS

Cette annexe fournit des phrases neutres et des modèles de messages que vous pouvez utiliser pour communiquer avec l'autre parent de façon respectueuse et factuelle. Utilisez ces modèles comme point de départ et adaptez-les selon votre situation.

Principes de communication neutre :
• Restez factuel et objectif
• Évitez les émotions et les jugements
• Concentrez-vous sur les besoins de l'enfant
• Soyez bref et clair
• Évitez les attaques personnelles

─────────────────────────────────────────────────────────────────────────────

LISTE DE PHRASES NEUTRES POUR SMS

CONFIRMATIONS

• "Confirmé pour [date] à [heure]."
• "D'accord pour [arrangement]."
• "Je confirme que [information]."
• "Merci pour la confirmation."

DEMANDES

• "Peux-tu confirmer [arrangement] ?"
• "Serait-il possible de [demande] ?"
• "Je te demande de confirmer [arrangement] avant [date]."
• "Merci de me confirmer [information]."

INFORMATIONS

• "[Nom de l'enfant] a [information]."
• "Je t'informe que [information]."
• "[Information] concernant [nom de l'enfant]."
• "Pour information : [information]."

CHANGEMENTS

• "Je dois modifier [arrangement] pour [raison brève]."
• "Serait-il possible de changer [arrangement] ?"
• "Je propose [alternative] pour [arrangement]."
• "Merci de me confirmer si [changement] est possible."

RÉPONSES

• "Merci pour l'information."
• "J'ai bien reçu [information]."
• "Je prends note de [information]."
• "Merci de m'avoir informé."

MODÈLES DE COURRIELS

MODÈLE 1 : CONFIRMATION D'ARRANGEMENT

Objet : Confirmation — [Sujet]

Bonjour [Nom],

Je vous confirme les arrangements suivants pour [nom de l'enfant] :

• [Arrangement 1]
• [Arrangement 2]
• [Arrangement 3]

Merci de votre collaboration.

Cordialement,
[Votre nom]

─────────────────────────────────────────────────────────────────────────────

MODÈLE 2 : DEMANDE DE CHANGEMENT D'HORAIRE

Objet : Demande de changement d'horaire — [Date]

Bonjour [Nom],

Je vous écris pour demander un changement d'horaire pour [date].

[Raison brève de la demande].

Je propose [alternative] comme solution.

Merci de me confirmer avant [date limite] si cette alternative vous convient.

Cordialement,
[Votre nom]

─────────────────────────────────────────────────────────────────────────────

MODÈLE 3 : COMMUNICATION D'INFORMATION

Objet : Information — [Sujet]

Bonjour [Nom],

Je vous informe que [information factuelle concernant l'enfant].

[Informations supplémentaires si nécessaire].

Merci de prendre note.

Cordialement,
[Votre nom]

─────────────────────────────────────────────────────────────────────────────

MODÈLE 4 : RÉPONSE À UNE DEMANDE

Objet : Réponse — [Sujet]

Bonjour [Nom],

Merci pour votre demande concernant [sujet].

[Votre réponse : acceptation, refus avec raison brève, ou contre-proposition].

[Informations supplémentaires si nécessaire].

Cordialement,
[Votre nom]

─────────────────────────────────────────────────────────────────────────────

MODÈLE 5 : NÉGOCIATION D'ARRANGEMENT

Objet : Proposition d'arrangement — [Sujet]

Bonjour [Nom],

Je vous écris pour proposer un arrangement concernant [sujet].

Je propose [proposition détaillée].

Qu'en pensez-vous ? Je suis ouvert(e) à discuter d'alternatives si cette proposition ne vous convient pas.

Merci de me répondre avant [date limite].

Cordialement,
[Votre nom]

─────────────────────────────────────────────────────────────────────────────

MODÈLE 6 : RÉPONSE À UNE PROVOCATION

Objet : Réponse — [Sujet]

Bonjour [Nom],

Merci pour votre message concernant [sujet].

Je préfère me concentrer sur les arrangements concernant [nom de l'enfant] et éviter les discussions personnelles.

Concernant [sujet spécifique], [réponse factuelle et brève].

Cordialement,
[Votre nom]

SCRIPTS POUR DIFFÉRENTES SITUATIONS

SCRIPT 1 : DEMANDE DE CHANGEMENT D'HORAIRE

Situation : Vous devez demander un changement d'horaire.

Message :
"Bonjour [Nom],

Je vous écris pour demander un changement d'horaire pour [date].

[Raison brève : ex. 'J'ai un rendez-vous médical important' ou 'J'ai une urgence professionnelle'].

Je propose [alternative : ex. 'de prendre [nom de l'enfant] le [date alternative] à la place'].

Merci de me confirmer avant [date limite] si cette alternative vous convient.

Cordialement,
[Votre nom]"

─────────────────────────────────────────────────────────────────────────────

SCRIPT 2 : COMMUNICATION D'INFORMATION SCOLAIRE

Situation : Vous devez communiquer une information concernant l'école.

Message :
"Bonjour [Nom],

Je vous informe que [nom de l'enfant] a [information : ex. 'une rencontre parents-professeurs le [date] à [heure]'].

L'école a demandé la présence des deux parents. Seriez-vous disponible pour y assister ?

Merci de me confirmer.

Cordialement,
[Votre nom]"

─────────────────────────────────────────────────────────────────────────────

SCRIPT 3 : NÉGOCIATION DE VACANCES

Situation : Vous devez négocier les arrangements de vacances.

Message :
"Bonjour [Nom],

Je vous écris pour discuter des arrangements de vacances pour [nom de l'enfant] pour [période].

Selon notre entente, [arrangement prévu].

Je propose [proposition : ex. 'de partager les vacances comme suit : première moitié avec moi, deuxième moitié avec vous'].

Qu'en pensez-vous ? Je suis ouvert(e) à discuter d'alternatives.

Merci de me répondre avant [date limite].

Cordialement,
[Votre nom]"

─────────────────────────────────────────────────────────────────────────────

SCRIPT 4 : RÉPONSE À UNE CRITIQUE

Situation : L'autre parent vous critique ou vous attaque.

Message :
"Bonjour [Nom],

Merci pour votre message.

Je préfère me concentrer sur les arrangements concernant [nom de l'enfant] et éviter les discussions personnelles.

Concernant [sujet spécifique si applicable], [réponse factuelle et brève].

Cordialement,
[Votre nom]"

─────────────────────────────────────────────────────────────────────────────

SCRIPT 5 : DEMANDE DE DOCUMENTATION

Situation : Vous devez demander des documents ou des informations.

Message :
"Bonjour [Nom],

Je vous écris pour demander [document ou information : ex. 'une copie du bulletin scolaire de [nom de l'enfant]'].

J'ai besoin de cette information pour [raison brève : ex. 'compléter ma déclaration de revenus' ou 'planifier les activités'].

Merci de me faire parvenir [document ou information] avant [date limite] si possible.

Cordialement,
[Votre nom]"

─────────────────────────────────────────────────────────────────────────────

SCRIPT 6 : CONFIRMATION D'URGENCE

Situation : Vous devez communiquer une urgence.

Message :
"Bonjour [Nom],

URGENT : [nom de l'enfant] a [situation : ex. 'besoin de soins médicaux urgents'].

[Informations nécessaires : ex. 'Je l'emmène à [lieu] maintenant. Je vous tiendrai informé(e) de la situation.'].

[Coordonnées si nécessaire].

[Votre nom]"

EXEMPLES CONCRETS (ANONYMISÉS)

EXEMPLE 1 : CONFIRMATION D'HORAIRE

Situation : Confirmation de l'horaire de garde pour la semaine.

Message :
"Bonjour Marie,

Je confirme l'horaire pour Sophie cette semaine :

• Lundi à vendredi : Avec moi (après l'école)
• Vendredi 18h : Transition à l'école
• Samedi et dimanche : Avec toi

Merci de confirmer.

Cordialement,
Jean"

─────────────────────────────────────────────────────────────────────────────

EXEMPLE 2 : DEMANDE DE CHANGEMENT

Situation : Demande de changement d'horaire pour un rendez-vous.

Message :
"Bonjour Marie,

Je dois demander un changement d'horaire pour vendredi prochain.

J'ai un rendez-vous médical important que je ne peux pas reporter.

Je propose de prendre Sophie jeudi soir à la place de vendredi soir, ou de la prendre samedi matin.

Merci de me confirmer avant mercredi si l'une de ces alternatives te convient.

Cordialement,
Jean"

─────────────────────────────────────────────────────────────────────────────

EXEMPLE 3 : COMMUNICATION D'INFORMATION MÉDICALE

Situation : Communication d'une information médicale importante.

Message :
"Bonjour Marie,

Je t'informe que Sophie a un rendez-vous chez le dentiste le 15 mars à 14h.

Le dentiste a recommandé [traitement ou information].

Je t'informe pour que tu sois au courant.

Cordialement,
Jean"

─────────────────────────────────────────────────────────────────────────────

EXEMPLE 4 : RÉPONSE À UNE PROVOCATION

Situation : L'autre parent vous critique dans un message.

Message original (provocateur) :
"Tu es toujours en retard. Tu ne respectes jamais les horaires. C'est inacceptable."

Réponse neutre :
"Bonjour Marie,

Merci pour ton message.

Je préfère me concentrer sur les arrangements concernant Sophie et éviter les discussions personnelles.

Concernant les horaires, je m'engage à être à l'heure pour les transitions futures.

Cordialement,
Jean"

─────────────────────────────────────────────────────────────────────────────

EXEMPLE 5 : NÉGOCIATION DE VACANCES

Situation : Négociation des vacances d'été.

Message :
"Bonjour Marie,

Je t'écris pour discuter des vacances d'été pour Sophie.

Selon notre entente, les vacances d'été sont partagées cette année.

Je propose de prendre Sophie du 1er au 15 juillet, et toi du 16 au 31 juillet.

Qu'en penses-tu ? Je suis ouvert à discuter d'alternatives si cette proposition ne te convient pas.

Merci de me répondre avant le 1er mai.

Cordialement,
Jean"

CONSEILS POUR UTILISER CES MODÈLES

ADAPTATION

Adaptez les modèles :
• Remplacez les [brackets] par vos informations
• Adaptez le ton selon votre situation
• Gardez le message bref et factuel
• Évitez les détails personnels

TIMING

Choisissez le bon moment :
• Envoyez les messages aux heures appropriées
• Donnez un délai de réponse raisonnable
• Évitez les messages en fin de semaine ou tard le soir (sauf urgence)

DOCUMENTATION

Documentez les communications :
• Gardez une copie de tous les messages
• Organisez les messages par date
• Utilisez un système de classement clair

LIMITES

Établissez des limites :
• Ne répondez pas aux provocations
• Ne participez pas aux discussions personnelles
• Concentrez-vous sur les arrangements concernant l'enfant
• Consultez un avocat si nécessaire
