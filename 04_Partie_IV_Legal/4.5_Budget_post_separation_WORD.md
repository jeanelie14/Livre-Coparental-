CHAPITRE 14 — BUDGET POST-SÉPARATION : PLAN FINANCIER, ASSISTANCE SOCIALE, ALLOCATIONS, AIDES LOCALES

─────────────────────────────────────────────────────────────────────────────

INTRODUCTION : BUDGET POST-SÉPARATION

La séparation a des conséquences financières importantes. Passer d'un ménage à deux revenus à un ménage à un revenu (ou à deux revenus séparés) peut créer des défis financiers significatifs. Cependant, avec une planification appropriée et une compréhension des ressources disponibles, il est possible de gérer votre situation financière et de planifier votre avenir.

Ce chapitre vous aidera à :
• Évaluer votre situation financière actuelle
• Créer un budget réaliste post-séparation
• Comprendre les aides disponibles au Québec
• Gérer la dette conjointe
• Créer un plan d'urgence financier
• Accéder aux ressources d'aide financière

Objectif : Vous donner les outils pour gérer votre situation financière après la séparation et planifier votre avenir financier.

─────────────────────────────────────────────────────────────────────────────

ÉVALUATION DE VOTRE SITUATION FINANCIÈRE

Avant de créer un budget, vous devez évaluer votre situation financière actuelle.

ÉTAPE 1 : IDENTIFIER VOS REVENUS

Revenus à inclure :
• Salaire : Revenus d'emploi (après impôts)
• Pension alimentaire : Pension alimentaire reçue (voir Chapitre 13)
• Allocations : Allocations familiales, crédits d'impôt, etc.
• Prestations : Prestations d'assurance-emploi, prestations d'invalidité, etc.
• Autres revenus : Revenus de placement, pension de retraite, etc.

Revenus mensuels :
• Calculez vos revenus mensuels totaux
• Incluez tous les revenus réguliers
• Notez les revenus irréguliers séparément

ÉTAPE 2 : IDENTIFIER VOS DÉPENSES

Dépenses à inclure :
• Logement : Loyer ou hypothèque, taxes, assurance, entretien
• Services publics : Électricité, chauffage, eau, internet, téléphone
• Nourriture : Épicerie, restaurants
• Transport : Voiture (paiement, assurance, essence), transport en commun
• Soins de santé : Médecin, dentiste, médicaments, assurance
• Éducation : Frais de scolarité, activités, fournitures
• Enfants : Vêtements, activités, soins, etc.
• Dettes : Paiements de dettes, cartes de crédit, prêts
• Autres : Assurance, épargne, loisirs, etc.

Dépenses mensuelles :
• Calculez vos dépenses mensuelles totales
• Incluez toutes les dépenses régulières
• Notez les dépenses irrégulières séparément

ÉTAPE 3 : CALCULER VOTRE SOLDE

Solde mensuel :
• Solde = Revenus - Dépenses
• Si le solde est positif : Vous avez un surplus
• Si le solde est négatif : Vous avez un déficit

Analyse :
• Identifiez les dépenses qui peuvent être réduites
• Identifiez les revenus qui peuvent être augmentés
• Planifiez comment équilibrer votre budget

GESTION DE LA DETTE CONJOINTE

Après la séparation, vous devez gérer les dettes que vous avez contractées ensemble.

TYPES DE DETTES CONJOINTES

Dettes communes :
• Hypothèque : Si vous avez acheté une maison ensemble
• Prêts : Prêts personnels, prêts auto, etc.
• Cartes de crédit : Cartes de crédit conjointes
• Dettes d'études : Si applicable
• Autres dettes : Autres dettes contractées ensemble

OPTIONS POUR GÉRER LES DETTES

Option 1 : Partage égal
• Partagez les dettes également entre les deux parties
• Chaque partie paie sa part
• Simple mais peut ne pas être équitable si les revenus sont différents

Option 2 : Partage proportionnel
• Partagez les dettes proportionnellement aux revenus
• La partie avec le revenu le plus élevé paie plus
• Plus équitable mais plus complexe

Option 3 : Vente des biens
• Vendez les biens (maison, voiture, etc.)
• Utilisez le produit de la vente pour payer les dettes
• Partagez le reste équitablement

Option 4 : Un parent assume les dettes
• Un parent assume toutes les dettes
• L'autre parent renonce à sa part des biens
• Peut être négocié dans l'entente de séparation

PROTECTION LÉGALE

Dans l'entente de séparation :
• Incluez une clause sur le partage des dettes
• Spécifiez qui paie quoi
• Spécifiez les conséquences si l'une des parties ne paie pas

[IMPORTANT] Même si l'entente spécifie qui paie quoi, les créanciers peuvent toujours vous poursuivre si vous êtes co-signataire. Consultez un avocat pour comprendre vos obligations.

PLAN D'URGENCE FINANCIER (3-6 MOIS)

Un plan d'urgence financier vous aide à faire face aux imprévus financiers.

POURQUOI UN PLAN D'URGENCE ?

Les imprévus peuvent inclure :
• Perte d'emploi
• Urgence médicale
• Réparation de voiture ou de maison
• Dépenses imprévues pour les enfants
• Retard dans le paiement de la pension alimentaire

COMMENT CRÉER UN PLAN D'URGENCE

Étape 1 : Calculez vos dépenses mensuelles essentielles
• Logement, services publics, nourriture, transport, soins de santé
• Multipliez par 3 à 6 mois
• C'est votre objectif d'épargne d'urgence

Exemple :
• Dépenses mensuelles essentielles : 2 500$/mois
• Objectif d'épargne (3 mois) : 7 500$
• Objectif d'épargne (6 mois) : 15 000$

Étape 2 : Créez un compte d'épargne séparé
• Ouvrez un compte d'épargne dédié à l'urgence
• Ne touchez pas à cet argent sauf en cas d'urgence
• Épargnez régulièrement (même de petits montants)

Étape 3 : Établissez un plan d'épargne
• Déterminez combien vous pouvez épargner chaque mois
• Fixez-vous un objectif réaliste
• Suivez vos progrès

Exemple :
• Objectif : 7 500$ (3 mois)
• Épargne mensuelle : 200$/mois
• Temps pour atteindre l'objectif : 37,5 mois (environ 3 ans)

Étape 4 : Réduisez les dépenses non essentielles
• Identifiez les dépenses que vous pouvez réduire
• Réduisez les dépenses non essentielles
• Utilisez l'argent économisé pour l'épargne d'urgence

Étape 5 : Augmentez vos revenus (si possible)
• Cherchez des opportunités d'augmenter vos revenus
• Travail à temps partiel, travail autonome, etc.
• Utilisez l'argent supplémentaire pour l'épargne d'urgence

POINTS CLÉS DU CHAPITRE

• Évaluer votre situation financière (revenus et dépenses) est la première étape pour créer un budget réaliste
• Les revenus moyens au Québec varient de 2 500$ à 4 000$/mois pour les familles monoparentales
• Les dépenses mensuelles types incluent : logement (800$-1 800$), services publics (200$-400$), nourriture (400$-700$), transport (300$-500$)
• Plusieurs aides sont disponibles au Québec : allocations familiales, crédits d'impôt, aide sociale, supplément pour enfant
• Un plan d'urgence financier (3-6 mois de dépenses essentielles) est essentiel pour faire face aux imprévus
• La gestion de la dette conjointe doit être incluse dans l'entente de séparation
• Utilisez le modèle de budget fourni pour créer votre propre budget et le réviser régulièrement

Action immédiate : Évaluez votre situation financière actuelle en utilisant le modèle de budget fourni. Identifiez vos revenus et vos dépenses, calculez votre solde, et créez un plan d'action pour équilibrer votre budget. Contactez les ressources d'aide financière si nécessaire.

Ressource : Pour les allocations familiales et les crédits d'impôt, contactez Revenu Québec au 1-800-267-7829 ou visitez revenuquebec.ca. Pour l'aide sociale, contactez le Centre local d'emploi au 1-877-767-8773 ou visitez emploiquebec.gouv.qc.ca.
