CHAPITRE 4.4 — CALCUL DE LA PENSION ALIMENTAIRE : PRINCIPES, OBLIGATIONS, TABLE QUÉBÉCOISE / FÉDÉRALE, EXEMPLES SIMPLES (SCÉNARIOS)

─────────────────────────────────────────────────────────────────────────────

INTRODUCTION : PENSION ALIMENTAIRE AU QUÉBEC

La pension alimentaire pour enfants est une obligation légale qui vise à assurer que les enfants bénéficient du même niveau de vie qu'ils auraient eu si leurs parents vivaient ensemble. Au Québec, le calcul de la pension alimentaire est basé sur des tables officielles qui tiennent compte des revenus des parents et du nombre d'enfants.

Ce chapitre vous aidera à :
• Comprendre les principes de base de la pension alimentaire
• Connaître vos obligations légales
• Comprendre la différence entre les tables québécoises et fédérales
• Calculer la pension alimentaire
• Comprendre comment modifier la pension
• Savoir quoi faire si l'autre parent ne paie pas

[IMPORTANT] Ce chapitre fournit des informations générales et des exemples. Le calcul exact de la pension alimentaire peut être complexe et dépend de nombreux facteurs. Consultez toujours un avocat ou utilisez un calculateur officiel pour un calcul précis.

─────────────────────────────────────────────────────────────────────────────

PRINCIPES DE BASE

La pension alimentaire pour enfants est basée sur plusieurs principes fondamentaux.

PRINCIPE 1 : L'INTÉRÊT DE L'ENFANT

L'intérêt de l'enfant est la priorité :
• La pension alimentaire vise à assurer le bien-être de l'enfant
• L'enfant a le droit de bénéficier du même niveau de vie qu'il aurait eu si ses parents vivaient ensemble
• Les besoins de l'enfant sont prioritaires sur les préférences des parents

PRINCIPE 2 : OBLIGATION DES DEUX PARENTS

Les deux parents ont une obligation :
• Les deux parents ont l'obligation légale de contribuer aux besoins de l'enfant
• La contribution est basée sur les revenus de chaque parent
• Le parent qui a la garde principale reçoit généralement la pension du parent qui n'a pas la garde principale

PRINCIPE 3 : PROPORTIONNALITÉ AUX REVENUS

La pension est proportionnelle aux revenus :
• Plus le revenu du parent payeur est élevé, plus la pension est élevée
• Plus le revenu du parent gardien est élevé, plus la pension peut être réduite (dans certains cas)
• La pension est calculée selon des tables officielles qui tiennent compte des revenus

PRINCIPE 4 : PARTAGE DES COÛTS

Les coûts sont partagés :
• Les coûts de base (nourriture, logement, vêtements, etc.) sont partagés
• Les coûts extraordinaires (activités, soins médicaux, éducation, etc.) peuvent être partagés séparément
• Le partage est généralement proportionnel aux revenus

─────────────────────────────────────────────────────────────────────────────

OBLIGATIONS LÉGALES

Les obligations légales concernant la pension alimentaire sont définies dans le Code civil du Québec et les Lignes directrices fédérales sur les pensions alimentaires pour enfants.

OBLIGATION LÉGALE DE PAYER

Le parent qui n'a pas la garde principale a l'obligation légale de payer une pension alimentaire :
• Cette obligation existe même si les parents ne sont pas mariés
• Cette obligation existe même si le parent n'a pas de droits de visite
• Cette obligation existe jusqu'à ce que l'enfant atteigne 18 ans (ou plus dans certains cas)

MONTANT MINIMUM

Il n'y a généralement pas de montant minimum :
• Le montant est calculé selon les tables officielles
• Même un revenu très faible peut entraîner une pension minimale
• Le montant peut être ajusté selon les circonstances

INDEXATION

La pension est généralement indexée :
• La pension est indexée annuellement selon l'indice des prix à la consommation (IPC)
• L'indexation est automatique si l'entente ou le jugement le prévoit
• L'indexation peut être modifiée si les revenus changent significativement

DURÉE DE L'OBLIGATION

L'obligation dure généralement jusqu'à 18 ans :
• L'obligation peut se poursuivre après 18 ans si l'enfant est aux études
• L'obligation peut se poursuivre si l'enfant a des besoins spéciaux
• L'obligation peut être modifiée si les circonstances changent

─────────────────────────────────────────────────────────────────────────────

TABLES QUÉBÉCOISES ET FÉDÉRALES

Il existe deux systèmes de calcul de la pension alimentaire au Canada : les tables québécoises et les tables fédérales.

QUAND UTILISER LES TABLES QUÉBÉCOISES

Utilisez les tables québécoises si :
• Les deux parents résident au Québec
• La procédure se déroule au Québec
• L'entente ou le jugement est rendu au Québec

Avantages :
• Les tables québécoises sont généralement plus avantageuses pour le parent payeur
• Les tables québécoises tiennent compte des particularités du système québécois (impôts, allocations, etc.)

QUAND UTILISER LES TABLES FÉDÉRALES

Utilisez les tables fédérales si :
• Un parent réside dans une autre province
• La procédure se déroule dans une autre province
• L'entente ou le jugement est rendu dans une autre province

Avantages :
• Les tables fédérales sont uniformes à travers le Canada
• Les tables fédérales peuvent être plus avantageuses dans certaines situations

DIFFÉRENCES PRINCIPALES

Tables québécoises :
• Tiennent compte des impôts québécois
• Tiennent compte des allocations québécoises
• Peuvent être différentes des tables fédérales

Tables fédérales :
• Uniformes à travers le Canada
• Tiennent compte des impôts fédéraux
• Peuvent être différentes des tables québécoises

[IMPORTANT] Consultez un avocat pour déterminer quelles tables s'appliquent à votre situation.

─────────────────────────────────────────────────────────────────────────────

CALCUL DE LA PENSION

Le calcul de la pension alimentaire est basé sur les revenus des parents et le nombre d'enfants.

REVENUS À CONSIDÉRER

Revenus à inclure :
• Salaire : Revenus d'emploi (T4)
• Revenus autonomes : Revenus d'entreprise, profession libérale, etc.
• Revenus de placement : Intérêts, dividendes, gains en capital, etc.
• Prestations : Prestations d'assurance-emploi, prestations d'invalidité, etc.
• Autres revenus : Pension de retraite, rentes, etc.

Revenus à exclure (généralement) :
• Allocations familiales : Généralement exclues
• Crédits d'impôt : Généralement exclus
• Aide sociale : Peut être exclue dans certains cas

Revenus ajustés :
• Les revenus sont ajustés pour tenir compte des impôts
• Les revenus sont ajustés pour tenir compte des déductions
• Les revenus sont ajustés pour tenir compte des dépenses professionnelles

FORMULE DE BASE

Calcul simplifié :

1. Déterminez le revenu annuel brut de chaque parent
   • Additionnez tous les revenus de chaque parent
   • Ajustez pour les impôts et déductions

2. Déterminez le revenu total des deux parents
   • Additionnez les revenus des deux parents

3. Déterminez le pourcentage de contribution de chaque parent
   • Parent A : Revenu A / Revenu total × 100%
   • Parent B : Revenu B / Revenu total × 100%

4. Consultez les tables officielles
   • Utilisez les tables québécoises ou fédérales selon votre situation
   • Trouvez le montant de base selon le revenu total et le nombre d'enfants

5. Calculez la pension de base
   • Pension de base = Montant selon les tables

6. Ajustez selon les revenus de chaque parent
   • Pension payée par Parent A = Pension de base × Pourcentage de contribution de Parent A
   • Pension payée par Parent B = Pension de base × Pourcentage de contribution de Parent B

7. Calculez la pension nette
   • Si Parent A a la garde principale : Pension nette = Pension payée par Parent B - Pension payée par Parent A
   • Si Parent B a la garde principale : Pension nette = Pension payée par Parent A - Pension payée par Parent B

[IMPORTANT] Cette formule est simplifiée. Le calcul réel peut être plus complexe et dépend de nombreux facteurs. Consultez un avocat ou utilisez un calculateur officiel pour un calcul précis.

AJUSTEMENTS POSSIBLES

1. Garde partagée :
• Si la garde est partagée (50/50), la pension peut être ajustée
• Le calcul peut être différent selon le temps passé avec chaque parent

2. Coûts extraordinaires :
• Les coûts extraordinaires (activités, soins médicaux, éducation, etc.) peuvent être partagés séparément
• Le partage est généralement proportionnel aux revenus

3. Besoins spéciaux :
• Si l'enfant a des besoins spéciaux, la pension peut être ajustée
• Les coûts supplémentaires peuvent être partagés séparément

4. Revenus très élevés ou très faibles :
• Si les revenus sont très élevés, la pension peut être ajustée
• Si les revenus sont très faibles, la pension peut être ajustée

5. Autres facteurs :
• Dépenses professionnelles
• Dettes
• Autres obligations financières

EXEMPLES CHIFFRÉS

Voici des exemples de calcul de pension alimentaire pour différents scénarios.

SCÉNARIO 1 : GARDE PRINCIPALE, REVENUS MOYENS

Situation :
• Parent A (garde principale) : Revenu annuel de 40 000$
• Parent B (pas de garde principale) : Revenu annuel de 60 000$
• Nombre d'enfants : 1 enfant
• Tables : Tables québécoises

Calcul :

1. Revenu total : 40 000$ + 60 000$ = 100 000$
2. Pourcentage de contribution :
   • Parent A : 40 000$ / 100 000$ = 40%
   • Parent B : 60 000$ / 100 000$ = 60%
3. Montant selon les tables (revenu total 100 000$, 1 enfant) : Environ 1 200$/mois
4. Pension payée par chaque parent :
   • Parent A : 1 200$ × 40% = 480$/mois
   • Parent B : 1 200$ × 60% = 720$/mois
5. Pension nette (Parent A a la garde principale) : 720$ - 480$ = 240$/mois

Résultat : Parent B paie 240$/mois à Parent A.

SCÉNARIO 2 : GARDE PARTAGÉE, REVENUS SIMILAIRES

Situation :
• Parent A (garde partagée) : Revenu annuel de 50 000$
• Parent B (garde partagée) : Revenu annuel de 50 000$
• Nombre d'enfants : 2 enfants
• Garde : 50/50 (garde partagée)
• Tables : Tables québécoises

Calcul :

1. Revenu total : 50 000$ + 50 000$ = 100 000$
2. Pourcentage de contribution :
   • Parent A : 50 000$ / 100 000$ = 50%
   • Parent B : 50 000$ / 100 000$ = 50%
3. Montant selon les tables (revenu total 100 000$, 2 enfants) : Environ 1 800$/mois
4. Pension payée par chaque parent :
   • Parent A : 1 800$ × 50% = 900$/mois
   • Parent B : 1 800$ × 50% = 900$/mois
5. Ajustement pour garde partagée : Comme la garde est partagée 50/50, la pension peut être réduite ou éliminée

Résultat : Avec une garde partagée 50/50 et des revenus similaires, la pension peut être réduite ou éliminée, selon les circonstances.

SCÉNARIO 3 : REVENUS TRÈS DIFFÉRENTS

Situation :
• Parent A (garde principale) : Revenu annuel de 30 000$
• Parent B (pas de garde principale) : Revenu annuel de 100 000$
• Nombre d'enfants : 1 enfant
• Tables : Tables québécoises

Calcul :

1. Revenu total : 30 000$ + 100 000$ = 130 000$
2. Pourcentage de contribution :
   • Parent A : 30 000$ / 130 000$ = 23%
   • Parent B : 100 000$ / 130 000$ = 77%
3. Montant selon les tables (revenu total 130 000$, 1 enfant) : Environ 1 500$/mois
4. Pension payée par chaque parent :
   • Parent A : 1 500$ × 23% = 345$/mois
   • Parent B : 1 500$ × 77% = 1 155$/mois
5. Pension nette (Parent A a la garde principale) : 1 155$ - 345$ = 810$/mois

Résultat : Parent B paie 810$/mois à Parent A.

SCÉNARIO 4 : REVENUS FAIBLES, 2 ENFANTS

Situation :
• Parent A (garde principale) : Revenu annuel de 25 000$
• Parent B (pas de garde principale) : Revenu annuel de 35 000$
• Nombre d'enfants : 2 enfants
• Tables : Tables québécoises

Calcul :

1. Revenu total : 25 000$ + 35 000$ = 60 000$
2. Pourcentage de contribution :
   • Parent A : 25 000$ / 60 000$ = 42%
   • Parent B : 35 000$ / 60 000$ = 58%
3. Montant selon les tables (revenu total 60 000$, 2 enfants) : Environ 900$/mois
4. Pension payée par chaque parent :
   • Parent A : 900$ × 42% = 378$/mois
   • Parent B : 900$ × 58% = 522$/mois
5. Pension nette (Parent A a la garde principale) : 522$ - 378$ = 144$/mois

Résultat : Parent B paie 144$/mois à Parent A.

SCÉNARIO 5 : REVENUS ÉLEVÉS, 3 ENFANTS

Situation :
• Parent A (garde principale) : Revenu annuel de 80 000$
• Parent B (pas de garde principale) : Revenu annuel de 120 000$
• Nombre d'enfants : 3 enfants

   • Parent A : 80 000$ / 200 000$ = 40%

5. Pension nette (Parent A a la garde principale) : 1 800$ - 1 200$ = 600$/mois

Résultat : Parent B paie 600$/mois à Parent A.

MODIFICATION DE LA PENSION

La pension alimentaire peut être modifiée si les circonstances changent.

QUAND MODIFIER LA PENSION

Circonstances qui justifient une modification :

1. Changement de revenus :
• Augmentation ou diminution significative des revenus d'un parent
• Perte d'emploi
• Changement d'emploi
• Retraite

2. Changement de garde :
• Changement de garde (garde principale → garde partagée, etc.)
• Changement du temps passé avec chaque parent

3. Changement des besoins de l'enfant :
• Besoins spéciaux
• Coûts extraordinaires (activités, soins médicaux, éducation, etc.)
• Changement d'âge (besoins différents selon l'âge)

4. Changement des circonstances :
• Nouvelle famille (nouveaux enfants)
• Dettes importantes
• Autres obligations financières

COMMENT MODIFIER LA PENSION

Option 1 : Entente entre les parents
• Les deux parents s'entendent sur une modification
• Rédigez une nouvelle entente ou modifiez l'entente existante
• Faites homologuer l'entente par le tribunal

Option 2 : Demande au tribunal
• Si les parents ne s'entendent pas, vous pouvez demander une modification au tribunal
• Vous devez démontrer qu'il y a eu un changement de circonstances
• Le tribunal examinera la demande et décidera

Procédure :
1. Rassemblez les preuves du changement de circonstances
2. Consultez un avocat
3. Déposez une demande de modification au tribunal
4. Présentez votre cas devant le juge
5. Le juge rend une décision

QUAND LA MODIFICATION N'EST PAS POSSIBLE

La modification n'est généralement pas possible si :
• Le changement de circonstances est temporaire
• Le changement de circonstances n'est pas significatif
• Le changement de circonstances était prévisible
• La demande est faite de mauvaise foi

QUE FAIRE SI L'AUTRE PARENT NE PAIE PAS

Si l'autre parent ne paie pas la pension alimentaire, vous avez plusieurs recours.

ACTIONS IMMÉDIATES

1. Contactez l'autre parent :
• Essayez de communiquer avec l'autre parent
• Demandez une explication
• Proposez un plan de paiement

2. Documentez les non-paiements :
• Tenez un registre des paiements manqués
• Gardez toutes les communications
• Documentez vos tentatives de contact

3. Consultez un avocat :
• Consultez un avocat pour connaître vos options
• Un avocat peut vous aider à faire exécuter la pension

RECOURS DISPONIBLES

1. Recouvrement par Revenu Québec :
• Revenu Québec peut recouvrer la pension alimentaire
• Revenu Québec peut retenir la pension sur le salaire du parent payeur
• Revenu Québec peut saisir les comptes bancaires, les prestations, etc.

Comment faire :
• Contactez Revenu Québec
• Fournissez une copie de l'entente ou du jugement
• Revenu Québec examinera votre demande

[GUIDE] Revenu Québec - Recouvrement de pension alimentaire
• Téléphone : 1-800-267-7829
• Site web : revenuquebec.ca

2. Exécution par le tribunal :
• Vous pouvez demander au tribunal de faire exécuter la pension
• Le tribunal peut ordonner des mesures d'exécution (saisie, etc.)

3. Saisie sur salaire :
• Vous pouvez demander une saisie sur le salaire du parent payeur
• Le tribunal peut ordonner que la pension soit retenue directement sur le salaire

4. Saisie de comptes bancaires :
• Vous pouvez demander une saisie des comptes bancaires du parent payeur
• Le tribunal peut ordonner la saisie

5. Saisie de biens :
• Vous pouvez demander une saisie de biens du parent payeur
• Le tribunal peut ordonner la saisie

CONSÉQUENCES POUR LE PARENT QUI NE PAIE PAS

Conséquences possibles :
• Intérêts : Intérêts sur les montants impayés
• Frais : Frais d'exécution
• Saisie : Saisie de salaire, comptes bancaires, biens
• Crédit : Impact sur le crédit
• Criminal : Dans certains cas, le non-paiement peut être criminel
