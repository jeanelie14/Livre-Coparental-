ANNEXE 2.B — MODÈLE DE JOURNAL D'INCIDENTS

─────────────────────────────────────────────────────────────────────────────

INSTRUCTIONS D'UTILISATION

1. Remplissez ce journal immédiatement après chaque incident
2. Soyez factuel et objectif (pas d'émotions, juste les faits)
3. Conservez ce journal dans un endroit sûr
4. Faites des copies si nécessaire

CONSEILS POUR REMPLIR LE JOURNAL

DESCRIPTION FACTUELLE

[À ÉVITER] Éviter : "Il était vraiment méchant"
[À PRÉFÉRER] Préférer : "Il a utilisé les mots suivants : [citer exactement]"

TÉMOINS

• Inclure nom complet et coordonnées si possible
• Notez la relation avec vous ou l'enfant

PREUVES

• SMS : Faire une capture d'écran
• Email : Conserver une copie
• Photos : Seulement si autorisées et légales
• Enregistrements : Vérifier la légalité au Québec

EFFET SUR L'ENFANT

• Décrire les comportements observés (pleurs, retrait, agressivité, etc.)
• Noter l'heure et la durée si applicable

ACTION ENTREPRISE

• Noter toutes les actions (appels, consultations, signalements)
• Conserver les preuves de ces actions (numéros d'appel, confirmations, etc.)
• Inclure les noms des personnes contactées et leurs coordonnées

CONSEILS SUPPLÉMENTAIRES

ORGANISATION

• Un incident par page : Facilite l'organisation et la consultation
• Ordre chronologique : Organisez les incidents par date
• Index : Créez un index pour retrouver rapidement les incidents
• Copies : Faites des copies régulières et conservez-les dans des endroits différents

CONFIDENTIALITÉ

• Conservez le journal en lieu sûr : Pas accessible à l'autre parent
• Faites des copies de sauvegarde : Cloud sécurisé, clé USB, copie papier chez une personne de confiance
• Protégez les informations : Ne partagez le journal qu'avec des professionnels de confiance

UTILISATION LÉGALE

• Consultez un avocat : Avant d'utiliser le journal en cour
• Assurez-vous de la légalité : Toutes les preuves doivent être légales
• Préparez votre dossier : Organisez le journal avec les autres preuves pour faciliter l'utilisation
