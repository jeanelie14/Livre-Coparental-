# Annexe — Modèle de journal d'incidents

---

## Instructions d'utilisation

1. Remplissez ce journal **immédiatement après chaque incident**
2. Soyez **factuel et objectif** (pas d'émotions, juste les faits)
3. **Conservez** ce journal dans un endroit sûr
4. **Faites des copies** si nécessaire

## Conseils pour remplir le journal

### Description factuelle
- ❌ Éviter : "Il était vraiment méchant"
- ✅ Préférer : "Il a utilisé les mots suivants : [citer exactement]"

### Témoins
- Inclure nom complet et coordonnées si possible
- Notez la relation avec vous ou l'enfant

### Preuves
- SMS : Faire une capture d'écran
- Email : Conserver une copie
- Photos : Seulement si autorisées et légales
- Enregistrements : Vérifier la légalité au Québec

### Effet sur l'enfant
- Décrire les comportements observés (pleurs, retrait, agressivité, etc.)
- Noter l'heure et la durée si applicable

### Action entreprise
- Noter toutes les actions (appels, consultations, signalements)
- Conserver les preuves de ces actions (numéros d'appel, confirmations, etc.)
